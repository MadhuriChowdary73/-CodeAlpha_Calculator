# Calculator App

A simple calculator built with HTML, CSS, and JavaScript.

## Features
- Basic operations: addition, subtraction, multiplication, division
- Delete (DEL) and Clear (C) buttons
- Decimal point support
- Display button

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Clone the repo
2. Open index.html in your browser
3. Start calculating!
